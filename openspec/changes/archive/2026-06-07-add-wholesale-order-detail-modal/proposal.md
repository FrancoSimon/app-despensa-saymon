## Why

Wholesalers need to review the products, quantities, prices, delivery date, and status of a previous order without leaving the portal.

## What Changes

- Include order items in the customer's wholesale order history query.
- Add a "Ver detalle" action to each order card in "Mis pedidos".
- Show order details in a modal with item breakdown and totals.

## Impact

- Affected specs: api-surface
- Affected code: wholesale query/types, wholesale order terminal

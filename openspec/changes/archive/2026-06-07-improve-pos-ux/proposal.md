## Why

The counter workflow needs to be faster and clearer for real selling conditions. Sellers benefit from keyboard shortcuts, focused search, mobile-friendly layout, and a confirmation step that reduces accidental sales.

## What Changes

- Improve product search priority and keyboard entry.
- Add keyboard shortcuts for search, payment selection, cart clearing, and sale confirmation.
- Add a clear sale confirmation modal before submitting.
- Improve POS cart responsiveness and sticky behavior on larger screens.
- Make blocked sale states clearer when the cash register is closed or account-current customer is missing.

## Impact

- Affected specs: api-surface
- Affected code: POS terminal

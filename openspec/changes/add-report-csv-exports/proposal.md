## Why

Admins can review report tables on screen, but only best-selling products can be exported. Daily sales, cash registers, and low-stock products also need CSV export for spreadsheet review.

## What Changes

- Make the CSV export button reusable for report tables.
- Add CSV export to daily sales, cash register period rows, and low-stock products.
- Keep the existing best-sellers export.

## Impact

- Affected specs: `reports-analytics`.
- Affected files: `components/reports`, `app/admin/reportes`, OpenSpec artifacts.
